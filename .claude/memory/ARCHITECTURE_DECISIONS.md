## Signal Engine пороги

- **Футбол**: минимум 4 из 6 сигналов
- **CS2**: минимум 6 из 10 сигналов (строже)
- **Порог EV**: > 10%, коэффициент ≥ 1.55, уверенность ≥ 52%

## Изоляция модулей (нарушать нельзя)

- CS2 код живёт ТОЛЬКО в `sports/cs2/`, публичный API через `sports/cs2/__init__.py`
- Нет зависимостей между видами спорта
- `sports/football/__init__.py` — только реэкспорт корневых модулей

## Масштабирование (план)

- 100 активных юзеров → мигрировать на `aiosqlite` (async SQLite)
- 250 юзеров → PostgreSQL на отдельном сервере
